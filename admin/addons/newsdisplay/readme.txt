News Display Module

This is a new module for displaying news items, this was orginal a built in feature in view.php but has since changed 
due for the increasing need to reduce code size and speed.

=======================================
Linking to this module
=======================================
link is as normal
view.php
will automaticly load this module you can change the category with c=category name or c=category id.

This module also supports the ability to filter and sort your news items in acending or decending order.

=======================================
Templates
=======================================
news style templates can be assigned with in the admin on top of this template you can also assign a table start and table end
template these appear at the top and bottom of your news and are totaly optional.

template names.
tablestart
tableend
