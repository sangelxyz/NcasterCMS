Arcticle view 1.5

This is a ncaster module for displaying a full page of news, it's simple and effective,
it supports

* Custom Tags, (Unlimited field or addon added tags).
* article styling

=======================================
Linking to an article
=======================================
Linking to a full news article: 
<a href="http://myhost.com/view.php?<!--url-->">Full News</a>
or
<a href="http://myhost.com/<!--url_dynamic-->">Full News</a>
Notice the diffence?

it's also possible to use the template over ride function to select a template manualy.
+ Example
<a href="http://myhost.com/view.php?<!--url-->&otemp=template name here">Full News</a>

Apart from these you can also use the <!--url_dynamic--> to link to a full news page
+ Example
<a href="http://myhost.com/<!--url_dynamic-->">Full News</a>
