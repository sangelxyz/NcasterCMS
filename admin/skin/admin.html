Ncaster 1.7 is the next evolution in the series it employs a new display engine frame work, Entity 2 scripting, Custom Field Filtering, image manipulation (watermarking, resizing) & Hubs (Article Relation system).

New Features.

  • Entity 2. Ncasters native template scripting language allows you to take the drivers seat and fully control the end result page, you can randomly select images, text use Rot13 to encode spoliers for games and decode it for those who want to see, create dynamic multicolored tables with over 30 modules included you can finaly create a site your prode of.
  • InherT. Ncaster features a full blown template editing solution you can edit each and every single modules template from styles & display templates each module also features a Inhertence system where each module can use the Universal module template or the category assigned template, InherT also features a Variable assign template which allows you to create custom tags on the fly that apply to all templates.
  • FieldNV/CFF. With FieldNV you have the power to create your own custom fields, this allows you to create rich and easy to edit sites such as Games, Music or genreal entertainment sites you can edit scores, release dates and more.
  • Assciative Hubs. One of the truly lacking features you will see in most premade cms systems is the abbility to relate articles or most importentily allowing the user sepreate frequently typed text. imagin if you had 500 news items about a game but suddenly the developer or game changes it's name, with a correctly set up ncaster site using hubs it whould only require the editing of one article instead of 500.
  • Spin WYSIWUG Html Editor. A light weight html editor is always handy when creating professional looking articles, you can do most of the basics such as bold, italic, underline, center, left, right and more.
  • Image Manuplation. There is no longer a need for stand along watermarking or resizing programs as ncaster and your server can do it for you, with ncaster you can customise the transperency color, compression level, icon and more.
  • Uploading. Ncaster supports both uploading with articles and stand-alone uploading, you can upload up to 99 items at once, each image can be resized & watermarked if needed.
  • BackUp. Ncaster features basic backing up of your ncaster database at just a few clicks of a button there is support for many compression formats such as zip, bzip & gzip to save download time.
  • Profiles. Profiles are anouther new addition to ncaster, each ncaster user can set there real name, msn, icq, yahoo, email contacts and even disable and enable the wysiwug html editor & N-code editor.
  • XML Output. RSS output is now supported by the included xmlfeed.php script.
