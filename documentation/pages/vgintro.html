· Project Ncaster 1.7.
Project ncaster 1.7 is brute combination of speed, flexability and ease of use it features:

Modular template engine:
The new Morlock template engine is a fully modular template engine it allows you to print and modifi variables on the fly and it also provides content encoding and caching (template and full page rendering).

Scritable templates: Entity 2.
A big part of the new template engine is ncaster 1.7's native template scripting language (Entity 2), To put it basicly entity allows you to modifi varables on the fly it features a module based engine with a whole host of included functions such as substring, lowercase, uppercase, colorswitch, ex_counter, articlelist, field exsists & field not exsists more modules will be avaliable for download in just a short time.

Two of the most powerfull entity functions is article list and colorswitch

Article List
Is a dynamic version of ncasters build list function optimised to run with minimal query ussage, it allows you to create formated lists of articles such as headlines, additinal news pages ect and place them any where in your template(s).

Color switch
While basic at hart it shows how easy yet powerfull entity 2 modules can be this color switch just switchs inbetween to colors which enables you to create multi-colored news items or used with modules for example you could have multi-colored tables.

a few other nice feature of entity 2 is the ability to modifi a verable with multiple functions example.
{$var|setvar|"Hey world"}
{$var|lower|""}
{* Current out put whould be hey world *}
{$var|substr|"3"}
{* now it's just hey*}

Each entity function can also have an ulimited amount of arguments & entity is persistent that means you can incremate numbers or store things in memorey to come back to later in anouther call examples of this is ex_counter which incremates for every time it's called which is great for adding numbers to tables.

New Design overhall & easy skining.
1.7 layout has been redesigned to be much faster loading, as well as the new design you will notice template managerment, article management ect have been updated to make your job easier it's now possible to remove or move 1 or more items.

Updated custom fields engine.
You can now create your own fields such as system,esrb, scores drop downs or ratio selects each can then be assigned to your custom fields.

Login rootines updated.
The login rootines have been completly re-wrote to support both cookie and no cookie logins.
