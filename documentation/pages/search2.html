Over View: Search 2 is a simple backend search engine, it finds articles by both subject and article.

Template styles used: template, tableend, tablestart

- Syntax run down.
- Search's including boolean.
- Basic search box html.
- Variables & tags for use in your templates.
- Supported features

Syntax .

Basic Search: ?load=search&find=grasshopper
Boolean Search: ?load=search&find=grass+er&boolean=y

Custom field filtering is also accepted to filter the end results.

Seaching .

Searching in ncasters search module is actualy quite easy simply type view.php?load=search&find=what to find from the browser, it's also possible to do a very basic boolean search such as view.php?load=search&find=what+to
that string will match the first string along with others. Search2 also supports the use of custom filters to filter the end result like in just about any other of ncasters modules.
