· Game List Module
Over View: The game list module simply renders xx amount of articles to screen.

Template styles used: template, tableend, tablestart

- Syntax run down.
- Variables & tags for use in your templates.
- Supported features


Syntax .

Display game list:
view.php?load=gamelist

Display game list with filters
view.php?load=gamelist&f_filtername=filtervalue

Variables & tags.

All standed news tags are supported as well as custom news tags you define.

Supported Features .

  • Entity 2
  • Custom Fields
  • Custom filtering
  • Fall back template system
  • Optional flat file templates
