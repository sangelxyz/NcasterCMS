Scripting with php.

Before we start, ncaster supports both <?php and <? to parse php regardless of weaver small tags are enabled by your server, however it does not support the quick echo tag <?= keep this in mind when scripting templates.

Basic hello world script.
<?
$var = 'hello world';
return $var;
?>

See how the variable is returned insted of printed, in order for you to display in place of the php starting and ending tags you need to return the data back to ncasters template parser, if you just print or echo to the screen your text will appear at the very top this is done to give you more control over your template.

<?
echo 'hello world';
?>

Will still however work. Since all data is returned and it's only possible to return a value you must append to your out put varaible what ever that may be so to add more text to a string as your script progresses you can do something like this

<?
$var = 'hello';
$var .= ' World';
return $var;
? >

<?

1. You can include any number of <?php, <? start and end statments with in any template including news stlyes.
2. Php is very fast however it does not have the in built function handling of entity, if you make a function in a news style it will cause an error becouse it will be redecleared for each load, however if you put it in your full page template it will work fine.

4. remember to return values that you want shown in the correct place.
