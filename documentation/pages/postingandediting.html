· Editing an article.
Ncaster has the ability to edit articles online in both nc-code or html or even both formats, each format has it's strengths many sites may enable both of them to take full advantage of each first i should make you aware of how each of these formats is handled before you begin posting articles.

Html Mode.
Html mode allows you to post any html code in the article area, this of couse has two draw backs ncaster can not filter it for any tags such as & >< or " this could present problems using the xml feed generater of possible even some browsers, to get arround this you must type the replacement for these tags &amp;, &gt; ect.

NC - Code Mode.
We highly recomend nc-code to be enabled, it's safer then html mode and does not have the draw back of invailed html.

Disabling html mode.
When you disable html mode all html special tags <> & " ect are converted in to printable tags, this allows you to actualy display html in your article or any code for that matter.
