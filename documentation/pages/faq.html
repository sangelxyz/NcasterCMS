Templates --

Can i make a template so when i change my url i can change all links by just changing a variable?
Yes, if entity is enabled you can use
{$var|setvar|"yoururl"}

then you use

{$var|nprint|""} or
<!$var$>

Where you want the value to be placed, you can then simple edit the setvar value when you change your url, you can also do the same thing in php using.

<?php
$mysite = 'your url';
?>

Then to print the value use.

<?php
echo $mysite;
?>
Both languages can be disabled so make sure, you enable the ones you wish to use. Entity can be enabled in it's own settings menu however php you will need to edit view.php file it's self.

If you want these variables to be avaliable to all templates, make a template and for it's title call it variables:global;

Can i use a scripting language to code my templates?
Yes, ncaster currently supports Entity2 and Php later on there maybe support for other 3rd party template scripting languages.

How do i assign a template?
You can assign a template using the assign menus in the settings/options menu or you can click edit on a category and assign the templates from there, when creating a category you also have the option to assign a template, beyond this you can also assign templates to modules using a diffrent method it's a naming system for your templates title put module:yourmodulename; that template is a universal template so it will apply to all categorys in that module if you just want to apply to one category you can use module:yourtemplatename{category:category id};, some modules also support style skining which allows you to style just about anything in the module to skin a part of a module you can use module:yourmodulename{style:stylename};

Does ncaster support xml out put?
It supports XML RSS feeds yes by the script xmlfeed.php.

How fast is ncaster? can i use it for a big site?
Ncaster is very fast it uses template caching, full page caching and part pre-rendering all 1st party modules have been taken in to consideration when optimising the database for speed, you should get very fast load times even with very large databases.

Can i use java or other coding languages with in my template for use in client side apps?
The template engine fully supports both java script and valid html on the one page, so you can use any language with in your templates.

Is it possible to filter Articles in the display script or in other modules by custom fields?
Ncaster supports a custom field filter system which allows you to filter by many diffrent fields. Heres some examples.
Field name: System Field values: Xbox, Pc, Gamecube
To show just the GameCube articles you whould use.
?f_System=GameCube
You can also filter by more then one field by doing
?f_System=GameCube&f_Score=3
This will print all GameCube articles with a Score of 3.

How do i enable entity?
In your admin go to settings/options > Entity settings, note by disabling entity template comments will not be removed nor will entity code, please keep this in mind when disabling the entity feature.

Hubs what do they do?
They act as a link, if you have ever used a computer hub for example all computers link to this one device so they can all comunicate with each other, Hubs in ncaster is much the same it allows entity for example to find related articles that are connected to there Hub. Hubs also act as a way to bind the hubs data soruce and the actual aritlce together this allows you to display both the hubs data and the articles data on the same page this can be done by simply using the <!$hub$> link tag in your templates
