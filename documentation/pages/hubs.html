Hubs.

In this section we will cover.
* Creating hubs
* Editing hubs.
* Deleting hubs.

Hubs Creation.
Hubs in a way are like categorys, they work like categorys you post & edit articles like you whould in a category however the diffrence is how they can be used, a hub is a data soruce for a category this allows you to seperate frequently used information in turn this saves time when posting and also saves even more time when editing as this method whould only require you to edit one page in order for the changers to apply to all assiciated articles.

Creating hubs :
Just like a category
To create a hub simply login to your admin click on settings > add a category.
then select yes where it says Make this category an assicative hub?.

Editing hubs :
To edit a hub simply login to your admin click on settings > edit a category, next you will given a list of possible category's/hubs to edit simply click the category you wish to edit.

Deleting hubs :
To Delete a hub simply login to your admin click on settings > remove a category.
Note: Deleting a hub will not remove any data, however once the category id is used it's not possible with in ncaster to regain it, to recreate a category you will need to use a tool like phpmyadmin.
