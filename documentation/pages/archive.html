· Archive Module
Over View: Shows a list of archives by month, when clicked it displays all the articles that where posted in that month.

Template styles used: For article list: template_list, tableend_list, tablestart_list For Month list: template, tableend, tablestart

- Syntax run down.
- Variables & tags for use in your templates.
- Supported features


Syntax .

Display game list:
view.php?load=archive

Display game list with filters (only avaliable when the month is clicked)
view.php?load=archive&finish=unixdate&start=unixdate&f_filtername=filtervalue

Variables & tags.

Month list
archive, url
Archive is the date of the archive and url is the link to the article archive.

All standed news tags are supported as well as custom news tags you define.

Supported Features .

  • Entity 2
  • Custom Fields
  • Custom filtering
  • Fall back template system
  • Optional flat file templates
