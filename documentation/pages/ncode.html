· N-Code (NC-Code)
Over View: N-Code or NC-Code is a variations of the popular ubbcode and vbbcode system, it works as a safe way of showing html with out actually typing html.

Tip: As of ncaster 1.7.2 you now have the option to set properties for specific tags, at the moment only [img] and [imageid] currently support this but others will at a later date.

Note: Make sure nc-code is enabled, otherwise the tags will not be processed.

- Syntex run down.
- Tag list.

Syntex .

1. Basic:
[tag]text here[/tag]

2. 2 Phase tag:
[tag=text here]text here[/tag]

3. Properties tag:
[tag arg1=ya arg2=ya2][/tag]

1. Basic is the most used type of tag.
2. Two phase tag is used mainly for links this allows you to give a description of the link as well as the actual link
3. With the last tag you can define properties for the tag, such as img height, width, border or even set a link.

 

Tags.

Tags: [tag][/tag] What it does. Useage:
img Displays the image from the location given. [img]url to image[/img]
[img border=2 height=10 width=7 class=image link=link]url to image[/img]
* This tag supports properties, this allows you to change the height, width ect, each property is optional.
imageid Displays a image from the id given [imageid]id of image[/imageid]
[imageid border=2 height=10 width=7 class=image link=link]id of image[/imageid]
* This tag supports properties, this allows you to change the height, width ect, each property is optional.
u Under line [u]text[/u]
i Italic [i]text[/i]
b Bold [b]text[/b]
quote Quote [quote]text[/quote]
li List [li][ol]text[/ol][/li]
* [li] is only to start a list, this must be followed with the ol or ul tags to create ordered or unordered lists.
ol Ordered list [li][ol]text[/ol][/li]
* [li] is only to start a list, this must be followed with the ol or ul tags to create ordered or unordered lists.
ul Un-orded list [li][ul]text[/ul][/li]
* [li] is only to start a list, this must be followed with the ol or ul tags to create ordered or unordered lists.
size Font size [size=size]text[/size]
color Font Color [color=color]text[/color]
font Font [font=font]text[/font]
table Start a table [table]text[/table]
td Start a column in a table [table][td]text[/td][/table]
tr Start a row [table][td][tr]text[/tr][/td][/table]
url Url Link [url]http://mylist.com[/url]
[url=http://mylist.com]my web page[/url]
email link to email address [email]michealo@ozemail.com.au[/email]

 

Most tags in the next release will have properties from which you can change.
