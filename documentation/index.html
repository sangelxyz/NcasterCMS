· Welcome to Project Ncaster: Zeus.
Project NCaster is a free revolutionary PHP/mySQL content management system featuring high customizability and flexibility. Some features include: the ability to create multiple programmable templates, advanced field and category customization, a comprehensive administration control panel, WYSIWYG html editor for formatting articles, the ability to create multiple pages for a single article and a built-in search engine. This script provides the power and efficiency to run a full-blown website any way you want.

Why use Ncaster.

Ncaster was designed from the ground up to be as flexible as possible, by in-corrupting exclusive and unique features such as FieldNV, CFF, Relational-Hubs, BuildList, Entity2 & the MorlockDisplay engine you are given unprecendented control over both input and output.

Template Caching
Large sites demand speed they need to load fast and feed out information to thousands, a dynamic system will just not cut it when it comes to the crunch, ncaster uses a template caching system that is extremely fast (as low as 0.004) at storing, retrieving and updating stored pre-rendered templates.

Un liner article association
Using the flexibility of Custom fields and the new Hubs feature you can now assign hubs to category's to create relational hubs, each of the fields stored with in the hub becomes a universal set for the article that is associated with it these fields only need to be changed in one place to apply to all articles associated, by assigning a hub article to a post you also create a Relational Link this can be used to call a hub to list all related articles to that hub, or give other articles inherited fields.

Verarible Modification
Ncaster 1.7 makes use of the entity 2 veriable modification system, it allows you to dynamic change values. Entity 2 is plugin based, new functionality can be added by just downloading new functions from the ncaster website.

Pre-Rendering
Fragment/part pre-rendering is supported in ncaster this allows you to pre-render parts of a page to make it load faster use less queries and in turn make you, your users and your server happy, at the moment ncaster only features the ability to pre-render lists of articles in descending order & by a single category this however enables you to create Spot light features, latest article lists or even an affiliate list. Each of these pre-rendered pages can be included in to your template by using the include syntax [include][/include] or if saved to database you can access them with {$var|compiled_list|"build list id"}.

Custom Fields
Some early news systems allowed you to have custom fields however they where limited by an amount and a single field type. With the ncaster FieldNV engine you can create an unlimited amount of new fields and define the type, size, value and sort order you can also create actual drop down lists & ratio selects with your own values an example of this would be if you ran a multiplatform game console site and you wanted to create a list of game systems gamecube, ps2, xbox, gameboy ect and maybe even a genre box or score box for your reviews all this and more can be done.

Search engine friendly
Some search engines do not index sites with dynamic content this in the past has been over come by modrewrite and various other hacks, ncaster features some modrewrite code as well as short urls.

Custom Field filtering
It's possible to filter articles by custom fields in ncaster which allows you to display the least amount of articles needed to give the user what they want, you can have up to 15 custom filters and 1 of those can be a multi condition filter.

Flexability
Ncaster was designed to be as UN-liner as possible you can set up a basic webpage or even an advanced website that shape shifts depending on values, images can be re-sized, articles can be edited, posted, deleted or moved you can create new fields new field types or even new category's on demand the possibility's are truly endless.

Module System
Modules in ncaster are extremely easy to create, maintain and edit each module can be coded as you would normally code a php script however the advantage of making a module for ncaster is that you need not create a template system and you can use it's fast array of classes and functions to create each module faster. Modules can be created for admins, standard loaded type or even additional processing layers.

Support.

FREE
Project Ncaster is fast, Free, Simple, Customizable, programmable & very flexible CMS which gives you total freedom to create your site, if ncaster does not support it, it can be coded in very easily using modules, process layers or even the ncaster admin. can be expanded to include new functions and features.

 

Installing Ncaster the easy way.

    Requirements Server:
    Mysql tested version 3.23 but should work on all 3x versions and above, PHP4 is a requirement.

    Recommended: Latest versions of Mysql and Php4, Linux operating system and apache web server.

    Requirements User: A java script compatible browser (netscape, mozilla, firebird, opera, ie) should all be fine, In most cases
    you could get away with a non java script browser except if you wish to use build list or the wysiwyg html editor

    Recommended: IE 5-IE6 to use WYSIWYG html editor, Mozilla compatibility is coming.

    Installing

  1. First you will need to download the ncaster zip from http://ncaster.cjb.net/
  2. Next unzip it’s contents using a unzip program. We recommend winzip & 7zip which can be found at www.winzip.com & www.7zip.org
  3. Next you will need to create a mysql database on your server this can be archived though a program such as PhpMyadmin, if you do not have access to create a new database or use an existing one you will need to contact your host provider.
  4. Next go to the directory you unzipped ncaster to edit with notepad (or anouther text editor) config.php, change lines 2 to 5 to reflect you’re mysql database information. Now save the file config.php.
  5. You will now need to get an ftp client to connect to your server we find cuteftp to be one of the best you can download a trial at www.cuteftp.com after you have downloaded cute ftp, enter your ftp details with in the program and connect to your server.
  6. After you have successfully connected to your server, upload all the ncaster files to your ftp host in there correct folders. NOTE: some hosts use the admin folder to get arround this problem just create a new folder and store ncaster with in that.
  7. Run install/install.php and follow the instructions
  8. Delete install folder when done. REQUIRED, for security reasons
  9. Ncaster is now installed on your server.

 

Upgrading Ncaster.

Ncaster 1.7 & 1.7.1 upgrade instructions
NOTE: BACK UP YOUR DATA BASE BEFORE ATTEMPTING AN UPDATE.

  1. Download the latest version of ncaster
  2. Unzip it's contents.
  3. Copy the folder & files with in admin/ to your computer to save as backup.
  4. Remove the admin admin/ folder from your server.
  5. Unzip the new ncaster to your hard drive
  6. Upload all the contents to your ftp except config.php, find the admin folder you backed up above and copy that config.php file to your new admin folder.
  7. Run, install/install.php and click upgrade.
  8. Now click ncaster 1.7.
  9. Your database will then be converted to the new stuture.
  10. Now delete Install folder

Ncaster 1.6.5 upgrade instructions
NOTE: BACK UP YOUR DATA BASE BEFORE ATTEMPTING AN UPDATE.

  1. Download the latest version of ncaster
  2. Unzip it's contents.
  3. Copy the folder & files with in admin/ to your computer to save as backup.
  4. Remove the admin admin/ folder from your server.
  5. Unzip the new ncaster to your hard drive
  6. Edit config.php with a text editor.
  7. Upload all the contents to your ftp.
  8. Find the old copy of config.php which is stored in admin/config.php.
  9. Rename it to config.bak.php and upload it to ncasters admin/ folder.
  10. It's now very importent to backup your mysql database in case something does not pan out the way you where hoping, you can do this by downloading a database management program from hotscripts.com.
    Note: some of the more recent CP's have phpmyadmin built in, alternatively you could ask your host to back it up if there is no other option.
  11. Run, install/install.php and click upgrade.
  12. Now if you old ncaster version was ncaster 1.6.5zeus (it should be if your reading these instructions) click that link.
  13. Your database will then be converted to the new stuture.
  14. Now delete Install folder

NOTE NCASTER 1.6.5 upgraders:
1. You will need to re-assign all your templates to there correct categorys
2. Tags with in these templates has also changed the tag sytex is now <!$tag$> and not <!--tag--> however it is possible to change it back to the original tag syntex by going to settings>options>render settings.

Ncaster 1.7b2 upgrade instructions
This way released on the ncaster forums about 1-2 months ago, not many people have it and even less use it becouse it was a beta.

Steps to upgrade

  1. Download the latest version of ncaster.
  2. Save a copy of config.php (localy) from your old ncaster install then remove your old ncaster install from the server make sure you remove all files and folders.
  3. Unzip ncaster 1.7
  4. Now edit config.php found in admin/ folder.
  5. Change the first 4/5 variables to reflect your mysql details.
  6. Upload ncaster 1.7 to your server.
  7. Run install (located at install/install.php)
  8. Now click the upgrade button.
  9. Click ncaster 1.7b2 to upgrade your old ncaster version.
  10. Upgrade is now complete.
